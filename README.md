# Elementy systemu
## Kanał kwantowy
Zadaniem kanału kwantowego jest dystrybucja symetrycznego klucza szyfrowania między dwoma użytkownikami (dalej A i B) z wykorzystaniem protokołu BB84. Ze względu na czynniki zewnętrzne, na które nie mieliśmy wpływu uzyskanie dostepu do prawdziwego kanału kwantowego nie powiodło się. Komponent ten jest symulowany przez program znajdujący się w repozytorium QKD należącym do organizacji BraKet-PBR.
